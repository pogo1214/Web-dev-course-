باب 1: HTML کا تعارف اور ویب کیسے کام کرتی ہے

❓ HTML کیا ہے؟

HTML کا مطلب ہے: HyperText Markup Language

یہ ایک ایسی زبان ہے جس کی مدد سے ہم ویب پیجز (Web Pages) بناتے ہیں۔ HTML ویب سائٹ کا "ڈھانچہ" (Structure) تیار کرتی ہے۔

📂 HTML کیا کام کرتا ہے؟

  • ویب پیج پر سرخیاں، پیراگراف، تصاویر اور لنکس لگاتا ہے
  • صارف کے لیے چیزیں نظر آنے والی بناتا ہے
  • ویب سائٹ کے مختلف حصوں کو ترتیب دیتا ہے

🧠 یاد رکھیں:

HTML خود "کام" (function) نہیں کرتا، وہ صرف صفحے کا ڈھانچہ بناتا ہے۔ اصل حرکت (design یا activity) CSS اور JavaScript سے آتی ہے۔

🖥️ ویب کیسے کام کرتا ہے؟ (سادہ زبان میں)

جب آپ Google Chrome یا کوئی بھی Browser کھول کر www.example.com لکھتے ہیں تو کیا ہوتا ہے؟

1
Browser آپ کے URL کو Internet پر بھیجتا ہے
2
Web Server سے HTML فائل واپس آتی ہے
3
Browser HTML کو پڑھتا ہے اور ایک خوبصورت صفحے کی شکل میں آپ کو دکھاتا ہے

📦 مثال سمجھیں:

جب کوئی شخص HTML میں یہ کوڈ لکھے:

<h1>میری پہلی ویب سائٹ</h1>
<p>یہ ایک پیراگراف ہے۔</p>

Browser اسے کچھ یوں دکھاتا ہے:

میری پہلی ویب سائٹ

یہ ایک پیراگراف ہے۔

🔁 HTML کی فائل کیسے بناتے ہیں؟

1
Notepad یا VS Code کھولیں
2
کوڈ لکھیں
3
Save کریں `.html` کے ساتھ
مثال: index.html
4
File پر double-click کریں، Browser میں کھل جائے گی!

📚 اس باب کے مشقیں

مشق 1:

ایک صفحہ بنائیں جس میں صرف "میری پہلی ویب سائٹ" heading ہو

مشق 2:

اس کے نیچے ایک پیراگراف ہو جس میں آپ اپنے بارے میں دو لائنیں لکھیں

مشق 3:

فائل کو `.html` میں محفوظ کریں اور Browser میں کھول کر دیکھیں

index.html

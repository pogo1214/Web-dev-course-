باب 12: CSS کے اہم Properties

رنگ، فونٹ، بیک گراؤنڈ، بارڈر اور دیگر اہم خصوصیات

1. رنگ (Colors)

CSS میں رنگ دینے کے تین طریقے ہیں:

/* رنگ کا نام استعمال کرتے ہوئے */
color: red;

/* Hex کوڈ استعمال کرتے ہوئے */
color: #ff0000;

/* RGB ویلیوز استعمال کرتے ہوئے */
color: rgb(255, 0, 0);
مثال:

یہ نیلا پیراگراف ہے

رنگوں کے مختلف طریقے:

نام سے: red

Hex سے: #ff5722

RGB سے: rgb(76, 175, 80)

عملی استعمال:

<p style="color: blue;">
  یہ نیلا پیراگراف ہے
</p>

2. بیک گراؤنڈ (Background)

بیک گراؤنڈ کو کنٹرول کرنے کے لیے CSS خصوصیات:

/* بیک گراؤنڈ رنگ */
background-color: yellow;

/* بیک گراؤنڈ تصویر */
background-image: url('image.jpg');

/* بیک گراؤنڈ دہرانا */
background-repeat: no-repeat;
مثال:

یہ ایک بیک گراؤنڈ کلر والا باکس ہے

عملی استعمال:

<div style="background-color: lightblue; padding: 10px;">
  یہ ایک بیک گراؤنڈ کلر والا باکس ہے
</div>

بیک گراؤنڈ تصویر:

گرڈینٹ بیک گراؤنڈ والا باکس

3. فونٹ (Fonts)

فونٹ کو کنٹرول کرنے کے لیے CSS خصوصیات:

خصوصیت تفصیل
font-family فونٹ کا نام (جیسے Arial, 'Nafees Nastaleeq')
font-size فونٹ کی سائز (px, em, rem میں)
font-weight فونٹ کی موٹائی (normal, bold)
font-style فونٹ کا انداز (italic, normal)

مثال:

یہ بولڈ اور خوبصورت فونٹ ہے

<p style="font-family: 'Courier New'; font-size: 18px; font-weight: bold;">
  یہ بولڈ اور خوبصورت فونٹ ہے
</p>

فونٹ سائز کا موازنہ:

12px فونٹ سائز

16px فونٹ سائز

24px فونٹ سائز

4. بارڈر (Border)

بارڈرز کو کنٹرول کرنے کے لیے CSS خصوصیات:

/* سادہ بارڈر */
border: 2px solid black;

/* بارڈر کی موٹائی، انداز اور رنگ الگ الگ */
border-width: 2px;
border-style: dashed;
border-color: green;

/* گول کونے */
border-radius: 10px;

مثال:

یہ ایک بارڈر والا باکس ہے

مختلف بارڈرز:

ٹھوس بارڈر (solid)
ڈیشڈ بارڈر (dashed)
ڈاٹڈ بارڈر (dotted)
<div style="border: 2px dashed red; padding: 10px;">
  یہ ایک بارڈر والا باکس ہے
</div>

5. Text کی Alignment

ٹیکسٹ کی سیدھ کو کنٹرول کرنے کے لیے:

text-align: left;
text-align: center;
text-align: right;
text-align: justify;
مثال:

مرکزی عنوان

مختلف Alignment:

دائیں جانب سیدھ (right)

درمیان میں سیدھ (center)

بائیں جانب سیدھ (left)

برابر سیدھ (justify) - یہ ٹیکسٹ دونوں جانب برابر ہوتا ہے۔ یہ خصوصیت لمبے پیراگراف کے لیے بہترین ہے۔

<h2 style="text-align: center;">
  مرکزی عنوان
</h2>

6. Inner & Outer Space

خصوصیت کام
padding اندر کی جگہ (عنصر اور اس کے مواد کے درمیان)
margin باہر کی جگہ (عنصر اور دیگر عناصر کے درمیان)
مثال:
margin اور padding استعمال ہوا ہے

عملی فرق:

باہر کی جگہ (margin)
اندر کی جگہ (padding)
مواد کا علاقہ
<div style="margin: 20px; padding: 15px; background-color: lightgray;">
  margin اور padding استعمال ہوا ہے
</div>

یاد رکھنے کے پوائنٹس

• CSS میں ہر لائن کے آخر میں semicolon (;) لگانا ضروری ہے

• CSS Style کو ہمیشہ curly braces ({}) میں رکھا جاتا ہے

• آپ CSS کو inline، internal یا external تینوں طریقوں سے استعمال کر سکتے ہیں

• جدید ویب ڈیزائن کے لیے external CSS سب سے بہتر طریقہ ہے

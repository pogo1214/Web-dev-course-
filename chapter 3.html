باب 13: CSS Layouts – Box Model اور Flexbox کا استعمال

1. Box Model کیا ہے؟

HTML کا ہر Element ایک box کی طرح ہوتا ہے۔

Content
Margin
Border
Padding
Content

Box Model میں 4 حصے ہوتے ہیں:

خصوصیت مطلب
margin Element کے باہر کی جگہ
border باکس کا بارڈر
padding Element کے اندر جگہ
content اصل مواد (text/image)

مثال:

<div style="margin: 20px; border: 2px solid black; padding: 10px;"> Box Model کا عملی مظاہرہ </div>

2. Flexbox کیا ہے؟

Flexbox سے آپ elements کو افقی یا عمودی لائن میں آسانی سے رکھ سکتے ہیں۔

Flexbox شروع کرنے کے لیے:

display: flex;

اہم Flexbox Properties:

خصوصیت کام
display: flex; Flexbox کو فعال کرتا ہے
flex-direction row (default) یا column
justify-content Horizontal alignment (مرکز وغیرہ)
align-items Vertical alignment

مثال:

<div style="display: flex; justify-content: space-around;"> <div style="background: red; padding: 20px;">Box 1</div> <div style="background: blue; padding: 20px;">Box 2</div> <div style="background: green; padding: 20px;">Box 3</div> </div>

Flexbox Layout Example: Sidebar + Content

<div style="display: flex;"> <div style="flex: 1; background: lightgray; padding: 20px;"> Sidebar </div> <div style="flex: 3; background: white; padding: 20px;"> Main Content </div> </div>

🧠 یاد رکھنے کی باتیں:
  • Flexbox کا استعمال Responsive Layout بنانے میں بہت آسان ہے
  • flex: 1, flex: 2, flex: 3 سے آپ سائز کنٹرول کر سکتے ہیں
  • Box Model سمجھنا ضروری ہے تاکہ spacing اور design ٹھیک رہے
  • Flexbox میں direction: rtl کا خیال رکھیں

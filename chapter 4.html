باب 14: HTML + CSS سے مکمل ویب پیج بنانا

ویب پیج میں شامل حصے:

  1. Header (اوپر والا حصہ)
  2. Navigation Bar (مینو)
  3. Main Content (درمیان کا حصہ)
  4. Sidebar (اگر چاہیں)
  5. Footer (نیچے والا حصہ)

مکمل HTML + CSS مثال:

<!DOCTYPE html> <html lang="ur"> <head> <meta charset="UTF-8"> <title>میری پہلی ویب سائٹ</title> <style> body { font-family: 'Noto Nastaliq Urdu', sans-serif; margin: 0; padding: 0; background-color: #f9f9f9; } header { background-color: #4CAF50; color: white; padding: 20px; text-align: center; } nav { background-color: #333; overflow: hidden; } nav a { float: right; display: block; color: white; text-align: center; padding: 14px 16px; text-decoration: none; } nav a:hover { background-color: #ddd; color: black; } .container { display: flex; padding: 20px; } .main { flex: 3; padding: 20px; background-color: white; } .sidebar { flex: 1; padding: 20px; background-color: #f2f2f2; } footer { background-color: #4CAF50; color: white; text-align: center; padding: 10px; position: relative; bottom: 0; width: 100%; } </style> </head> <body> <header> <h1>میری پہلی ویب سائٹ</h1> </header> <nav> <a href="#">رابطہ</a> <a href="#">سروسز</a> <a href="#">ہمارے بارے میں</a> <a href="#">ہوم</a> </nav> <div class="container"> <div class="main"> <h2>خوش آمدید</h2> <p>یہ ویب سائٹ HTML اور CSS کی مدد سے بنائی گئی ہے۔</p> </div> <div class="sidebar"> <h3>مزید پڑھیں</h3> <ul> <li>HTML کیا ہے؟</li> <li>CSS کا استعمال</li> <li>ویب ڈیزائن کے اصول</li> </ul> </div> </div> <footer> <p>© 2025 میری پہلی ویب سائٹ</p> </footer> </body> </html>

خوش آمدید

یہ ویب سائٹ HTML اور CSS کی مدد سے بنائی گئی ہے۔

یہ ایک بنیادی ویب پیج کا ڈھانچہ ہے جسے آپ اپنی ضروریات کے مطابق تبدیل کر سکتے ہیں۔

مزید پڑھیں

  • HTML کیا ہے؟
  • CSS کا استعمال
  • ویب ڈیزائن کے اصول

  • HTML کے مختلف sections: header, nav, main, sidebar, footer
  • CSS سے styling، layout اور structure

Mini Project 1: Personal Info Card

🎨 مقصد:

اپنا نام، تصویر، معلومات اور لنک دکھانے والا ایک خوبصورت کارڈ بنائیں۔

💡 آپ اس میں کیا سیکھیں گے؟

فیچر سیکھا گیا طریقہ
Shadow box-shadow کا استعمال
Rounded Borders border-radius
Buttons & Links Styling with hover effect
Text & Image Alignment and spacing

🖥️ مکمل Code:

<!DOCTYPE html> <html lang="ur"> <head> <meta charset="UTF-8"> <title>میرا کارڈ</title> <style> body { font-family: 'Noto Nastaliq Urdu', sans-serif; display: flex; justify-content: center; align-items: center; height: 100vh; background-color: #f0f0f0; } .card { background-color: white; width: 300px; border-radius: 15px; box-shadow: 0 0 10px rgba(0,0,0,0.2); text-align: center; padding: 20px; } .card img { width: 100px; height: 100px; border-radius: 50%; } .card h2 { margin-top: 10px; } .card a { display: inline-block; margin-top: 10px; text-decoration: none; color: white; background-color: #4CAF50; padding: 8px 16px; border-radius: 8px; } .card a:hover { background-color: #45a049; } </style> </head> <body> <div class="card"> <img src="https://via.placeholder.com/100" alt="میری تصویر"> <h2>آپ کا نام</h2> <p>ویب ڈویلپر، اردو لرنر</p> <a href="#">مزید دیکھیں</a> </div> </body> </html>

پیش منظر:

آپ کا نام

ویب ڈویلپر، اردو لرنر

مزید دیکھیں
🧠 مشق کا سوال:

خود سے کوشش کریں:

  • تصویر بدلیں
  • اپنا نام لکھیں
  • بٹن کے رنگ کو تبدیل کریں
  • نیچے ایک اور بٹن ڈالیں (جیسے: "Contact Me")

تفہیم <div> ٹیگ اور HTML سیمانٹک ٹیگز

ویب ڈویلپمنٹ میں بنیادی فرق، استعمال کے طریقے، اور بہترین طریقہ کار

<div> ٹیگ کیا ہے؟

<div> کا مطلب ہے "ڈویژن" یعنی کسی صفحے کے حصہ یا بلاک کو الگ کرنا۔

  • یہ ایک غیر سیمانٹک (Non-Semantic) ٹیگ ہے
  • یہ خود سے نہیں بتاتا کہ اندر کیا مواد ہے
  • بنیادی طور پر ایک کنٹینر کے طور پر کام کرتا ہے
  • CSS سٹائلنگ اور JavaScript کے لیے بہترین ہے
  • ویب پیج کے لیے ڈھانچہ (structure) بنانے میں مدد کرتا ہے

سیمانٹک ٹیگز کیا ہیں؟

سیمانٹک ٹیگز اپنے نام سے ہی اپنا مقصد ظاہر کرتے ہیں۔

  • یہ براؤزرز اور سرچ انجنز کو مواد کی ساخت سمجھنے میں مدد کرتے ہیں
  • ویب سائٹ کی رسائی (accessibility) بہتر بناتے ہیں
  • کوڈ کو پڑھنا اور سمجھنا آسان بناتے ہیں
  • جدید HTML5 کا اہم حصہ ہیں
  • مثال: <header>, <footer>, <article>, <section>

<div> کب استعمال کریں؟

  • جب آپ کو کئی عناصر کو ایک ساتھ گروپ کرنا ہو
  • جب آپ CSS سے سٹائل دینا چاہتے ہوں
  • جب JavaScript کے ذریعے منطق (logic) لاگو کرنی ہو
  • جب کوئی مناسب سیمانٹک ٹیگ دستیاب نہ ہو
  • پیج کے لیآؤٹ کی تشکیل کے لیے
ٹپ: جدید HTML میں پہلے مناسب سیمانٹک ٹیگ استعمال کریں، پھر <div> کا استعمال کریں

سیمانٹک ٹیگز اور <div> کا موازنہ

کام سیمانٹک ٹیگ <div> متبادل
صفحہ کا سربراہ <header> <div class="header">
صفحہ کا پاورقی <footer> <div class="footer">
مرکزی مواد <main> <div class="main">
مضامین کا حصہ <article> <div class="article">
سیکشن <section> <div class="section">
ناویگیشن <nav> <div class="nav">
اضافی معلومات <aside> <div class="sidebar">

<div> کے ساتھ کوڈ مثال

<div class="profile">
  <div class="header">
    <h2>میرا تعارف</h2>
  </div>
  <div class="content">
    <p>میرا نام علی ہے اور میں HTML سیکھ رہا ہوں۔</p>
  </div>
  <div class="footer">
    <p>ای میل: ali@example.com</p>
  </div>
</div>

نتیجہ:

میرا تعارف

میرا نام علی ہے اور میں HTML سیکھ رہا ہوں۔

ای میل: ali@example.com

سیمانٹک ٹیگز کے ساتھ کوڈ مثال

<article>
  <header>
    <h2>میرا تعارف</h2>
  </header>
  <section>
    <p>میرا نام علی ہے اور میں HTML سیکھ رہا ہوں۔</p>
  </section>
  <footer>
    <p>ای میل: ali@example.com</p>
  </footer>
</article>

نتیجہ:

میرا تعارف

میرا نام علی ہے اور میں HTML سیکھ رہا ہوں۔

ای میل: ali@example.com

نتیجہ اور بہترین طریقہ کار

جدید ویب ڈویلپمنٹ میں سیمانٹک ٹیگز کو ترجیح دیں کیونکہ یہ:

• ویب سائٹ کی رسائی (accessibility) بہتر بناتے ہیں
• سرچ انجن آپٹیمائزیشن (SEO) میں مددگار ہیں
• کوڈ کو پڑھنا اور برقرار رکھنا آسان بناتے ہیں

<div> ٹیگ استعمال کریں جب:

• آپ کو CSS یا JavaScript کے ذریعے سٹائلنگ یا فعالیت شامل کرنی ہو
• کوئی مناسب سیمانٹک ٹیگ دستیاب نہ ہو
• آپ عناصر کو گروپ کرنا چاہتے ہوں

بہترین طریقہ: سیمانٹک ٹیگز کے ساتھ ڈھانچہ بنائیں، اور <div> کو سٹائلنگ اور لیآؤٹ کے لیے استعمال کریں!

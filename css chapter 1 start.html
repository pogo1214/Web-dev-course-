باب 11: CSS کی بنیاد – HTML ویب سائٹ کو خوبصورت بنانا

📘 CSS کیا ہے؟ (Cascading Style Sheets)

CSS کا کام ہے HTML ویب سائٹ کو خوبصورت، رنگین اور ترتیب سے بنانا۔

📌 CSS کا کام کیا ہے؟

CSS HTML Tags کو style دیتا ہے جیسے:

  • رنگ (color)
  • فونٹ (font)
  • بارڈر (border)
  • بیک گراؤنڈ (background)
  • جگہ (margin, padding)
  • ترتیب (layout)

🧠 CSS کیسے کام کرتا ہے؟

CSS HTML عناصر کو منتخب کرکے ان پر مختلف خصوصیات (properties) لاگو کرتا ہے۔

🧪 CSS لکھنے کے 3 طریقے:

طریقہ کہاں لکھتے ہیں؟ استعمال
1️⃣ Inline CSS HTML Tag کے اندر style="" جلدی test کے لیے
2️⃣ Internal CSS <style> tag میں HTML کے <head> میں ایک ہی صفحہ کے لیے
3️⃣ External CSS الگ .css فائل پوری ویب سائٹ کے لیے

✅ مثال 1: Inline CSS

<p style="color: green; font-size: 20px;">یہ سبز پیراگراف ہے۔</p>

✅ مثال 2: Internal CSS

<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: blue;
      text-align: center;
    }
    p {
      font-family: 'Arial';
      font-size: 18px;
    }
  </style>
</head>
<body>

  <h1>میری ویب سائٹ</h1>
  <p>یہ ایک خوبصورت پیراگراف ہے۔</p>

</body>
</html>

✅ مثال 3: External CSS

HTML فائل:

<head>
  <link rel="stylesheet" href="style.css">
</head>

style.css فائل:

body {
  background-color: #f0f0f0;
}
h1 {
  color: purple;
}

🔠 CSS میں Color کیسے دیتے ہیں؟

طریقہ مثال نمونہ
نام سے color: red;
Hex Code سے color: #ff0000;
RGB سے color: rgb(255, 0, 0);

✨ CSS کے کچھ ضروری Properties

Property کام مثال
color ٹیکسٹ کا رنگ color: blue;
background بیک گراؤنڈ رنگ یا تصویر background: #f0f0f0;
font-size فونٹ کی سائز font-size: 16px;
font-family فونٹ کا نام font-family: Arial;
text-align Text کی alignment text-align: center;
padding اندر کی جگہ padding: 10px;
margin باہر کی جگہ margin: 20px;
border بارڈر لگانا border: 1px solid black;
width/height چوڑائی اور لمبائی دینا width: 100px;

📌 CSS کا فائدہ:

  • ایک ہی جگہ سے ساری ویب سائٹ کا ڈیزائن کنٹرول
  • HTML اور Style الگ الگ رہتے ہیں
  • ویب سائٹ خوبصورت اور پروفیشنل لگتی ہے
  • کوڈ کا دوبارہ استعمال ممکن ہوتا ہے
  • صفحات کا سائز کم ہوتا ہے
  • مختلف ڈیوائسز کے لیے مختلف اسٹائلز

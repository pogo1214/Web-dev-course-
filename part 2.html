باب 2: HTML Structure اور بنیادی ٹیگز

🧱 HTML Structure (ڈھانچہ) کیا ہوتا ہے؟

جب بھی آپ کوئی HTML فائل بناتے ہیں تو اس کا ایک مخصوص ڈھانچہ (structure) ہوتا ہے، جس میں خاص ٹیگز (Tags) ہوتے ہیں۔ یہ ٹیگز ویب صفحے کی بنیاد بناتے ہیں۔

✅ HTML کا مکمل بنیادی ڈھانچہ

<!DOCTYPE html> <html> <head> <title>میری ویب سائٹ</title> </head> <body> <h1>خوش آمدید!</h1> <p>یہ میری پہلی HTML فائل ہے۔</p> </body> </html>

🔍 تفصیل کے ساتھ وضاحت:

Tag اردو میں مطلب کام
<!DOCTYPE html> یہ HTML5 کا اعلان کرتا ہے براؤزر کو بتاتا ہے کہ یہ HTML5 دستاویز ہے
<html> ویب صفحے کا مرکزی کنٹینر ساری HTML اسی کے اندر لکھی جاتی ہے
<head> پوشیدہ معلومات اس میں عنوان، لنکس، اور CSS جیسے مواد ہوتا ہے
<title> ویب صفحے کا نام یہ نام browser کے tab میں نظر آتا ہے
<body> صفحے کا ظاہر ہونے والا حصہ جو بھی چیز صارف کو دکھانی ہو وہ یہاں آتی ہے

📝 یاد رکھنے والی بات:

  • <html> اور </html> کے درمیان پورا ویب صفحہ ہوتا ہے۔
  • <head> وہ حصہ ہے جو نظر نہیں آتا، مگر ضروری ہوتا ہے۔
  • <body> وہ حصہ ہے جو صارف کو براؤزر میں نظر آتا ہے۔

💡 مثال: ایک سادہ صفحہ

<!DOCTYPE html> <html> <head> <title>میرا ہوم پیج</title> </head> <body> <h1>السلام علیکم</h1> <p>یہ میرا پہلا HTML ویب پیج ہے۔</p> </body> </html>

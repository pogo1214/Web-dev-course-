باب 3: Text Tags – Headings, Paragraphs, Line Break

🏷️ 1. Headings Tags (<h1> سے <h6>)

Headings (سرخیاں) ویب پیج میں عنوانات (Titles) کو ظاہر کرنے کے لیے استعمال ہوتی ہیں۔

✅ HTML میں 6 قسم کی Headings ہوتی ہیں:

<h1>یہ سب سے بڑی heading ہے</h1> <h2>یہ دوسری بڑی heading ہے</h2> <h3>یہ تیسری بڑی heading ہے</h3> <h4>یہ چوتھی بڑی heading ہے</h4> <h5>یہ پانچویں بڑی heading ہے</h5> <h6>یہ سب سے چھوٹی heading ہے</h6>

📌 یاد رکھیں:

  • SEO (گوگل سرچ) میں <h1> بہت اہم ہوتی ہے
  • ہر صفحے میں صرف ایک <h1> ہونا بہتر ہے

📝 2. Paragraph Tag (<p>)

Paragraph ایک مکمل جملہ یا پیراگراف ظاہر کرتا ہے۔

✅ مثال:

<p>میرا نام علی ہے۔ میں ویب ڈویلپمنٹ سیکھ رہا ہوں۔</p>

📌 فائدہ:

ایک بات کو الگ پیراگراف میں لکھنا واضح اور آسان بناتا ہے

🔁 3. Line Break (<br>)

Line Break ایک لائن کو توڑنے کے لیے استعمال ہوتا ہے، یعنی ایک ہی پیراگراف میں نئی لائن شروع کرنے کے لیے۔

✅ مثال:

<p>یہ پہلی لائن ہے۔<br>یہ دوسری لائن ہے۔</p>

📌 نوٹ:

  • <br> self-closing tag ہے، اس کا اختتام نہیں ہوتا
  • صرف جب آپ کو نئی لائن بنانی ہو تو استعمال کریں

🌟 مکمل مثال:

<!DOCTYPE html> <html> <head> <title>Text Tags Example</title> </head> <body> <h1>ہیڈنگ 1</h1> <h2>ہیڈنگ 2</h2> <p>یہ ایک پیراگراف ہے۔<br>یہ نئی لائن ہے اسی پیراگراف میں۔</p> </body> </html>

🧪 مشقیں:

مشق 1:

ایک HTML فائل بنائیں جس میں:

  • <h1> میں آپ کی پسندیدہ کتاب کا نام ہو
  • دو <p> ہوں: ایک میں کتاب کا تعارف اور دوسرے میں آپ کی رائے
  • ایک <br> لگا کر کسی لائن کو توڑیں

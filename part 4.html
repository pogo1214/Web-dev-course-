باب 4: Links اور Images (لنک اور تصاویر)

🔗 1. HTML Links – <a> Tag

Link (رابطہ) کسی دوسرے صفحے، ویب سائٹ، فائل یا سیکشن پر جانے کے لیے استعمال ہوتا ہے۔

✅ Syntax:

<a href="رابطہ">لنک کا نام</a>

✅ مثال:

<a href="https://www.google.com">گوگل پر جائیں</a>

📌 وضاحت:

  • href="..." وہ جگہ ہے جہاں لنک لے کر جائے گا
  • <a> وہ Tag ہے جو کلک ہونے والا حصہ بناتا ہے

🔒 Extra: New Tab میں کھولنا

<a href="https://www.google.com" target="_blank">گوگل نئے ٹیب میں</a>

🖼️ 2. HTML Images – <img> Tag

Image (تصویر) دکھانے کے لیے HTML میں <img> استعمال ہوتا ہے۔

✅ Syntax:

<img src="تصویر کا لنک" alt="تصویر کا متبادل متن">

✅ مثال:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools Logo">

📌 وضاحت:

  • src میں تصویر کا لنک یا فائل کا نام آتا ہے
  • alt وہ متن ہے جو تب نظر آتا ہے جب تصویر لوڈ نہ ہو

📐 Optional: Image کی چوڑائی اور اونچائی

<img src="image.jpg" width="300" height="200">

🔁 دونوں کو ملا کر ایک مثال:

<!DOCTYPE html> <html> <head> <title>Links and Images</title> </head> <body> <h1>میری پسندیدہ ویب سائٹ</h1> <a href="https://www.wikipedia.org" target="_blank">وکی پیڈیا پر جائیں</a> <h2>میری تصویر</h2> <img src="https://via.placeholder.com/300x200" alt="میری تصویر" width="300" height="200"> </body> </html>

🧪 مشقیں:

مشق 1:

ایک HTML صفحہ بنائیں جس میں:

  • ایک لنک ہو جو YouTube.com پر جائے
  • ایک تصویر ہو جو کسی آن لائن URL سے لی گئی ہو

مشق 2:

تصویر کا alt اور width لازمی استعمال کریں

مشق 3:

Browser میں چلا کر دیکھیں

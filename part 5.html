باب 5: Lists in HTML – Ordered & Unordered Lists

📋 1. Unordered List – بغیر ترتیب والی فہرست

Unordered List وہ فہرست ہوتی ہے جس میں آئٹمز (Items) گول نقطے (•) کی شکل میں ظاہر ہوتے ہیں۔ ترتیب ضروری نہیں ہوتی۔

✅ Syntax:

<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul>

🖼️ براؤزر میں ایسا نظر آئے گا:

  • HTML
  • CSS
  • JavaScript

🔢 2. Ordered List – ترتیب والی فہرست

Ordered List وہ فہرست ہوتی ہے جس میں آئٹمز (Items) نمبر یا حروف کے ساتھ ظاہر ہوتے ہیں۔

✅ Syntax:

<ol> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ol>

🖼️ براؤزر میں ایسا نظر آئے گا:

  1. HTML
  2. CSS
  3. JavaScript

📝 List Tags کی وضاحت:

Tag کام
<ul> Unordered list کو ظاہر کرتا ہے
<ol> Ordered list کو ظاہر کرتا ہے
<li> List کا ہر ایک آئٹم

💡 خاص باتیں:

  • <ul> اور <ol> دونوں میں صرف <li> آئٹمز لکھے جاتے ہیں
  • Lists nested (یعنی ایک کے اندر دوسری) بھی بنائی جا سکتی ہیں

🎯 مکمل مثال:

<!DOCTYPE html> <html> <head> <title>HTML Lists</title> </head> <body> <h1>Programming Languages</h1> <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> <li>Python</li> <li>Java</li> </ul> <h2>My Daily Routine</h2> <ol> <li>Wake up</li> <li>Study</li> <li>Practice coding</li> <li>Sleep</li> </ol> </body> </html>

🧪 مشقیں:

مشق 1:

ایک unordered list بنائیں جس میں 5 programming languages ہوں

مشق 2:

ایک ordered list بنائیں جس میں آپ کا روزمرہ کا 4-step schedule ہو
